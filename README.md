# CosmoNode Feedback

Welcome to the CosmoNode feedback repository.

This repository is used for **bug reports, feature requests, and questions** related to CosmoNode. If you encounter an issue or have ideas for improving the tool, please open an issue.

---

## Documentation

Full documentation, guides, and tutorials are available here:

**Documentation**  
https://uumutunal.github.io/CosmoNode_Manual/

The documentation includes:

- Getting started guides
- Node reference and explanations
- Runtime API usage

---

## Before Opening an Issue

Please check the following before creating a new issue:

- Make sure you are using the **latest version of CosmoNode**
- Check if the issue has **already been reported**
- Include **clear reproduction steps** if reporting a bug

Providing detailed information helps resolve issues faster.

---

## Issue Types

You can use this repository for:

### 🐞 Bug Reports
Report problems, crashes, incorrect behavior, or compatibility issues.

### ✨ Feature Requests
Suggest new nodes, improvements, or workflow enhancements.

### ❓ Questions
Ask questions about usage, workflows, or runtime integration.

---

## Bug Report Guidelines

When reporting a bug, please include:

- **CosmoNode version**
- **Unity version**
- **Steps to reproduce the issue**
- **Expected behavior**
- **Actual behavior**
- Screenshots or example graphs if possible

---

## Feature Requests

When suggesting a feature, it helps to include:

- The problem you are trying to solve
- A suggested solution
- An example use case

---

Thank you for helping improve CosmoNode.
